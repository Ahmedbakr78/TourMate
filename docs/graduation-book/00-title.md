## Submission Statement

This graduation book documents the design, implementation, and evaluation of **TourMate**, a web-based tourism platform that enables tourists to plan trips, assign guides and drivers, track movement in real time through a polling mechanism, vote on points of interest, and report lost-and-found items. The work presented herein is the original effort of the authors and accurately reflects the completed, working source code contained within the accompanying repository.

## Document Structure

This book is organized into ten technical chapters accompanied by a unified modeling language (UML) appendix:

1. Introduction
2. System Architecture
3. Database Design
4. Backend Implementation
5. Frontend Implementation
6. UI/UX Design
7. System Testing
8. Quality Assurance
9. Conclusion and Future Work
10. Table of Contents

The UML diagrams — use-case, class, sequence, activity, and entity-relationship — are provided as a separate appendix under `docs/uml/` and are derived directly from the implemented endpoints and data models.
